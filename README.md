# PerfumeLab Pro ERP/POS v2.0

## نظام ERP/POS احترافي للعطور

### المميزات
- واجهة عربية كاملة (RTL)
- نظام عملات متعدد مع أسعار صرف
- نقاط بيع احترافية
- إدارة مخزون متعدد المستودعات
- نظام خزينة متعدد العملات
- تصنيع العطور (وصفات + مواد خام)
- برنامج ولاء العملاء
- تقارير ديناميكية
- سجل تدقيق كامل

### التشغيل
```bash
pip install -r requirements.txt
python src/main_app.py
```

### الهيكل
```
src/
  core/         - النواة (قاعدة البيانات، الخدمات، المديرون)
  modules/      - الوحدات (POS، المخزون، الخزينة، التصنيع...)
  ui/           - واجهة المستخدم (عناصر، سمات، حوارات)
  resources/    - الموارد (خطوط، صور، ترجمة)
```
